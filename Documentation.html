Documentation 

  • !help ➡ Shows the "Help" command
  • !play [song/url] ➡ Plays a song from YouTube
  • !pause ➡ Pauses the currently playing song
  • !resume ➡ Resumes a paused song
  • !stop ➡ Makes the bot leave the channel
  • !queue ➡ Shows the queue
  • !volume ➡ Changes the volume (0-100) caution default is 5, 100 is extremely loud
  • !nowplaying (!np) ➡ Shows the song that is currently being played
  • !loop ➡ Loops a song
  • !avatar [@user] ➡ Shows the avatar of mentioned user
  • !beautiful [@user] ➡ Shows 'this is beautiful' meme with user avatar
  • !di [text] ➡ The bot says your [text]
  • !fact ➡ Shows a random fact
  • !kiss [@user] ➡ Kiss a user
  • !trigger ➡ Trigger yourself
  • !wanted [@user] ➡ Wanted poster for @user
  • !wasted [@user] ➡ GTA V 'Wasted' screen 
  • !calculate [math operation] ➡ Resolves an Arithmetic problem
  • !covid [Country] ➡ Shows COVID-19 information
  • !ping ➡ Shows the bot latency in miliseconds
  • !wiki [Term] ➡ Looks for a term in Wikipedia
